## Melon loader Mods for Last Epoch

Originally was focus on tooltips stuff but now includes basically all of my personnal stuff in the DEV folder and other mods that I find interesting to share
You can find the release for the following mods on NexusMods, but you can also build them yourself if you want to check the code or if you want to use the latest version that is not yet released.

# [Fallen_Improved_Tooltips](https://www.nexusmods.com/lastepoch/mods/10)
Shows which filter rule matched and if unique/set items are already owned. 

# [Fallen_Auto_Enabler](https://www.nexusmods.com/lastepoch/mods/25)
Enable nearby Shrines/Caches/Chests etc without having to manually click on them, includes in game settings!
